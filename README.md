# skybox

Public issue tracker for **skybox**: code-repository indexer, graph store, hybrid search, and MCP server for the sky stack. A single static binary.

Development happens in a private repository; this is where bug reports, feature requests, and beta feedback live. When reporting, include your version and platform where relevant.

Part of the [sky stack](https://github.com/skylence-be), alongside [skyline](https://github.com/skylence-be/skyline).
